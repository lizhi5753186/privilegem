项目介绍：
 =
Privilegem 是一个基于SpringMVC + Mybatis + MySQL 后端权限管理系统。IDE使用Intellij IDEA进行开发的.该项目也是我这边第一个学习Java的项目，通过该项目大家可以快速掌握SpringMVC +Mybatis的使用。
